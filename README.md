# WinWhisper · Windows 本地离线语音转写工具

**WinWhisper** 是一款为 Windows 用户打造的本地离线语音转写工具。  
支持音频 / 视频拖拽导入，可一键生成 **文本（TXT）**、**字幕文件（SRT / VTT）**。  
全程离线处理，不上传任何资料，适合对隐私要求较高的个人与企业。

➡ **官方网站（下载 & 功能介绍）**  
https://codezen-dev.github.io/winwhisper-models/

---

## ✨ 核心特性

- **本地离线 · 隐私优先**  
  转写全过程在本机完成，不上传音频文件到云端服务器。  
  适用于会议纪要、访谈、课堂录音、企业内部资料等敏感场景。

- **拖拽即用的转写体验**  
  将音频 / 视频文件拖入窗口即可开始转写，无需复杂配置。  
  自动调用内置 ffmpeg 进行格式转换。

- **支持 TXT / SRT / VTT 导出**  
  可直接用于剪辑软件、字幕制作与文档整理。

- **历史记录管理**  
  自动保存转写结果，可快速查看和复用内容。

- **内置 Whisper 模型**  
  开箱即用，无需手动下载模型。
